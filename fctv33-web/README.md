# FCTV33 Web - Versión Web del App FCTV33 Premium

## Descripción
Versión web de la aplicación FCTV33 Premium para streaming de canales de TV.

## Tecnologías
- React + TypeScript
- Vite (build tool)
- HLS.js (streaming)
- TailwindCSS

## Dominios de la API (detectados en la APK)
- rbtv77.com - Servidor principal
- API endpoints: vip1.openaicommunities.shop

## Estructura detectada de la APK
- HomeActivity: Pantalla principal
- PlayerActivity: Reproductor
- ActivationActivity: Sistema de licencias

## Setup
```bash
npm install
npm run dev
```
